What is all this?

The Tengu Open Source UI is based on the Neo4j Browser. It uses the same mechanism as the Neo4j Browser: the command shell. Use the editor bar up above ↑ to enter client-side commands. Each command will produce a "frame" like this one in the result stream.

Use the :helpcommand to learn about other topics.

New to Tengu? Try one of the guides to learn the basics.

Usage: :help <topic>
Topics: :help commands
Guides: :play tengu
Example models: :tengu model create storm
:tengu model create spark
:tengu model create cityot
