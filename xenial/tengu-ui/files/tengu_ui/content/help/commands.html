Commands

The editor bar up above ↑ understands a few client-side commands, which begin with a :. In contrast to the Neo4j Browser, the Open Source Tengu UI does not support (yet) typing commands without a colon.

Information: :help help:help play
Tengu: :help tengu hauchiwa:help tengu model
Sojobo connectivity: :help server  (This will drastically change in future releases)
Editor: :help clear
Command history: :help history
