Tengu Hauchiwa

The  :tengu hauchiwa command lets you manage Tengu Hauchiwas. An Hauchiwa is used to control one or more Tengu models. Via an Hauchiwa a user can easily connect to the services of its deployed models.

The :tengu hauchiwa create <name> command lets you create an empty Hauchiwa. You can use this empty Hauchiwa to manually deploy Tengu models. The :tengu hauchiwa status <name|url> command shows the status of the Hauchiwa and the Tengu model(s) associated with this Hauchiwa.

Usage: :tengu hauchiwa <action>
Actions: :tengu hauchiwa create <name> :tengu hauchiwa status <name|url>
